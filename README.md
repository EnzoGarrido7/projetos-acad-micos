# projetos-acad-micos
Projetos realizados em ambiente acadêmico.
Este repositório reúne projetos desenvolvidos ao longo da graduação, envolvendo atividades práticas, estudos e aplicações nas áreas de programação, banco de dados, engenharia de software e tecnologia. O objetivo é registrar a evolução acadêmica, aplicar conceitos aprendidos em aula e compartilhar soluções desenvolvidas durante os estudos.
